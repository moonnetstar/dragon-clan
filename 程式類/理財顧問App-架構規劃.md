# 理財顧問 App — 架構規劃
# 工程師布魯斯 撰寫
# 日期：2026-06-03

## 目標
讓勞勞能在手機上測試修正的理財顧問 App

## 技術選型
- **框架：Flutter**（跨平台，iOS + Android 一套程式碼）
- **語言：Dart**
- **後端：Firebase**（免費方案就夠用）
- **資料來源：Yahoo Finance API**（免費即時股價）

## 核心功能

### 1. 持股管理
- 新增/編輯/刪除持股
- 記錄買入價格、股數、日期
- 即時計算損益、報酬率

### 2. 即時報價
- 台股即時價格（Yahoo Finance API）
- 漲跌幅、成交量
- 技術指標：MA5/MA20/MA60、RSI、MACD

### 3. 投資組合分析
- 總市值、總損益、總報酬率
- 各股佔比圓餅圖
- 每日/每月績效走勢

### 4. 買賣建議（桃樂絲 AI）
- 根據技術面給出建議
- 超買/超賣警示
- 停損停利提醒

### 5. 新聞與資訊
- 台股重要新聞
- AI 產業動態
- 黃金價格

### 6. 設定
- 預設持股清單
- 通知設定
- 主題（深色/淺色）

## 頁面結構
```
首頁（投資組合總覽）
├── 持股列表
│   ├── 持股詳情
│   └── 新增/編輯持股
├── 報價查詢
│   ├── 搜尋股票
│   └── 技術分析圖表
├── 新聞資訊
├── 設定
└── 關於
```

## 開發階段
- Phase 1：持股管理 + 即時報價（MVP）
- Phase 2：投資組合分析 + 圖表
- Phase 3：AI 買賣建議
- Phase 4：新聞推播 + 通知
