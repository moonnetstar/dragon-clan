## 一、為什麼要學這些？

勞勞的投資原則：**保守穩健、保護本金**

學這些指標不是要你變成操盤手，而是：
- 知道什麼時候該注意風險
- 不會因為恐慌而在低點殺出
- 有紀律地執行停損和獲利了結

## 二、最重要的一個指標：RSI

### 白話文
RSI 告訴你：「最近漲太多了還是跌太多了？」

### 怎麼看

| RSI 值 | 狀態 | 白話 | 操作建議 |
|--------|------|------|---------|
| **> 80** | 🔥 嚴重超買 | 漲過頭了，大家搶著買 | ✅ 考慮賣出 |
| **70-80** | ⚠️ 超買 | 漲很多，開始注意 | 🔍 密切觀察 |
| **30-70** | 正常 | 正常波動 | 📌 持有觀察 |
| **20-30** | ⚠️ 超賣 | 跌很多，開始注意 | 🔍 觀察是否止跌 |
| **< 20** | ❄️ 嚴重超賣 | 跌過頭了，大家拋售 | ⚠️ 不要恐慌殺低 |

### 實際案例
- 和碩 RSI 84 → 嚴重超買 → 該考慮賣了 ✅
- RSI 50 → 中性，不偏多也不偏空
- RSI 25 → 超賣，但不要急著買，先觀察

### 記住一句話
**RSI > 70 要注意，RSI < 30 不要慌**

## 三、移動均線（MA）— 趨勢方向

### 白話文
均線告訴你：「最近平均價格是多少？股價在平均之上還是之下？」

### 怎麼看

| 均線名稱 | 天數 | 用途 |
|----------|------|------|
| **MA5** | 5 天 | 這一週的平均價（極短期） |
| **MA20** | 20 天 | 這一個月的平均價（短期） |
| **MA60** | 60 天 | 這三個月的平均價（中期） |
| **MA120** | 120 天 | 這半年的平均價（長期） |
| **MA240** | 240 天 | 這一年的平均價（超長期） |

### 判斷方式（最簡單的記法）

**多頭排列（上漲中）：**
```
股價 > MA5 > MA20 > MA60
```
→ 像樓梯一樣往上爬 📈

**空頭排列（下跌中）：**
```
股價 < MA5 < MA20 < MA60
```
→ 像滑梯一樣往下滑 📉

**黃金交叉（買進訊號）：**
- MA20 從下方往上穿越 MA60
→ 🟢 趨勢可能要反轉向上

**死亡交叉（賣出訊號）：**
- MA20 從上方往下穿越 MA60
→ 🔴 趨勢可能要反轉向下

### 實際案例
- 和碩股價 101 元，MA20 是 95 元，MA60 是 90 元
- 101 > 95 > 90 → 多頭排列，趨勢向上 ✅

## 四、MACD — 趨勢強度

### 白話文
MACD 告訴你：「漲勢還有沒有力氣？還是在加速？」

### 怎麼看

| MACD 信號 | 白話 | 建議 |
|-----------|------|------|
| MACD > 0 且在上升 | 漲勢有力，繼續漲 | 🟢 持有或加碼 |
| MACD > 0 但在下降 | 漲勢減弱，要沒力了 | ⚠️ 注意反轉 |
| MACD < 0 且在下降 | 跌勢持續，不要接刀 | 🔴 觀望 |
| MACD < 0 但在上升 | 跌勢減緩，可能反彈 | 🟡 觀察 |

### 記住一句話
**MACD > 0 偏多，MACD < 0 偏空**

## 五、成交量 — 人氣指標

### 白話文
成交量告訴你：「有多少人在交易這檔股票？」

### 怎麼看

| 組合 | 白話 | 訊號 |
|------|------|------|
| **放量上漲** | 很多人追買，漲勢有力 | 🟢 好現象 |
| **放量下跌** | 很多人拋售，危險 | 🔴 快跑 |
| **縮量上漲** | 沒人追了，漲勢可能到頂 | ⚠️ 注意 |
| **縮量下跌** | 恐慌過了，可能止跌 | 🟡 觀察 |

### 記住一句話
**量是價的先行指標。有量才有價。**

## 六、停損的重要性

### 勞勞的停損原則

| 類型 | 停損比例 | 說明 |
|------|---------|------|
| **單檔停損** | 7% | 個股虧損達 7% 立即出場 |
| **整體組合** | 5% | 全部持股單日跌 5% 主動提醒 |

### 為什麼要停損？

```
假設本金 100 萬
虧 50% → 剩 50 萬
要回本 → 需要漲 100%（不是 50%！）

虧 10% → 剩 90 萬
要回本 → 只需漲 11.1%

虧 7% → 剩 93 萬
要回本 → 只需漲 7.5%
```

**結論：小虧容易回本，大虧很難翻身。停損是保本金的關鍵。**

## 七、除息基本知識

### 什麼是除息？
公司把盈餘用現金發放給股東，股價會「扣除」配息金額。

### 重要日期
| 日期 | 說明 |
|------|------|
| **公告日** | 公司宣布配息金額 |
| **除息日** | 股價扣除配息金額（當天下跌是正常的） |
| **發放日** | 現金股利入帳 |

### 和碩 2026 除息資訊
- 除息日：2026/06/15（週一）
- 現金股利：4.01 元/股
- 1000 股可領：4,010 元
- 發放日：2026/07/09

### 填息 vs 貼息
- **填息**：除息後股價漲回除息前水準 → ✅ 賺到股利 + 價差
- **貼息**：除息後股價沒漲回去 → ⚠️ 只賺到股利，帳面虧損

### 和碩過去填息紀錄
- 5 年內完全填息：3 次（60% 機率）
- 平均填息天數：42 個交易日（約 2 個月）

## 八、龍族給勞懶的懶人投資口訣

1. **RSI > 70 要注意，RSI < 30 不要慌**
2. **跌破停損價，出場不停留**
3. **不要 All-in，分批進場最安全**
4. **獲利股票分批鎖利，不要一次賣光**
5. **虧損股票嚴格停損，不要攤平**
6. **量是價的先行指標，有量才有價**
7. **除息前注意棄權息賣壓**
8. **紀律比貪心值錢**
