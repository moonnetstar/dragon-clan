## 雙切邏輯

### 預設模式（日常）
- 各成員使用本地 Ollama 小模型
- 快速回應、省 token、低成本

### 深度模式（觸發條件）
以下關鍵詞自動切換大模型：
- 「深層思考」「深度思考」「deep think」
- 「討論」「分析」「評估」
- 「複雜」「困難」「需要想」
- 「策略」「規劃」「決策」

### 重要規則
- **成員深度模型統一使用 sgemma4-26B-A4B（本地 26B）**
- **不使用 owl-alpha**（owl-alpha 是桃樂絲專屬）
- 桃樂絲維持 openrouter/owl-alpha 不動

---

## 手動切換指令

```bash
# 切換成員
hermes profile owl-alpha-1    # 華爾特
hermes profile owl-alpha-2    # 布魯斯
hermes profile owl-alpha-3    # 艾蒂兒
hermes profile default         # 回桃樂絲

# 切換後需要重啟 Gateway
hermes gateway restart
```
